# Roadmap

## Fase 1 - Fondamenta

- MCP server Python minimale
- tool sicuri base
- registry agenti
- router task rule-based
- documentazione standard

## Fase 2 - Orchestrazione

- execution plan strutturato
- validazione pre-run
- output JSON standard per agenti
- logging decisionale

## Fase 3 - GitHub workflow

- lettura issue/PR
- generazione branch operativi
- proposta patch controllata
- apertura PR con checklist

## Fase 4 - Voice front-end

- webhook/orchestratore per comandi vocali
- routing task da prompt parlato
- sintesi risultato finale

## Fase 5 - Hardening

- policy sicurezza tool
- sandbox locale
- audit log
- test e quality gate
